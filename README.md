# Submitty

Submitty is an open source course management, assignment submission, exam, and grading system from the [Rensselaer Center for Open Source Software (RCOS)](https://rcos.io/), the [Department of Computer Science](https://science.rpi.edu/computer-science), and [Rensselaer Polytechnic Institute](https://www.rpi.edu/).

This repository is a GitHub copy of the Submitty project. The main project organization is hosted at [github.com/Submitty](https://github.com/Submitty), and full user and developer documentation is available at [submitty.org](https://submitty.org/).

## Live View

## What the Project Does

Submitty helps instructors and students manage programming and course assignments. It supports assignment submission, automated grading, manual grading workflows, exams, course management, and tools for running a class at scale.

## Features

- Course and assignment management.
- Student assignment submission workflows.
- Automated grading support.
- Manual grading and review tools.
- Exam and assessment support.
- Instructor and student course workflows.
- Open source development model with contributor documentation.

For the official feature list, see [submitty.org/index/features](https://submitty.org/index/features).

## Tech Stack

Submitty is a large full-stack web application. Its repository includes web application code, grading infrastructure, developer tooling, scripts, tests, and deployment support. See the official developer documentation for the current architecture and setup details.
